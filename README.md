## 🚀 Getting Started

No build tools, no dependencies to install. Clone and open.

```bash
git clone https://github.com/shantha26/<repo-name>.git
cd <repo-name>
open index.html        # or just double-click it
```

### Deploying

Drop `index.html` and `Shantha_Lakshmi_S_Resume.pdf` into the same folder on any static host:

- **Netlify** — drag and drop the folder, or connect the GitHub repo
- **GitHub Pages** — push to a repo, enable Pages on the `main` branch
- **Vercel** — import the repo, no build command needed

- **Account → Security** — your live domain is on the allowed-origins list
